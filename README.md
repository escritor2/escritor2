## 👤 Sobre mim

Olá, sou **Gabriel Ferreira**, desenvolvedor focado em transformar problemas reais em soluções técnicas robustas. Minha atuação combina **automação**, **IoT**, **processamento de dados**, **React Native** e **Vue.js**, com atenção especial para arquiteturas limpas, integração entre hardware e software, e interfaces que entregam valor de forma consistente.

Atualmente, estou construindo um perfil técnico orientado a projetos, observabilidade e automação de processos — desde sensores e fluxos de dados até aplicações web/mobile com experiência de uso bem pensada.

<div align="center">

| Foco técnico | Como aplico |
|---|---|
| **Automação** | Crio fluxos repetíveis, integrações e scripts que reduzem esforço manual e aumentam confiabilidade. |
| **IoT** | Conecto dispositivos, protocolos e dados para transformar sinais físicos em informação acionável. |
| **Processamento de dados** | Organizo, transformo e visualizo dados para apoiar decisões técnicas e de negócio. |
| **Mobile** | Desenvolvo experiências com React Native, priorizando performance, usabilidade e manutenção. |
| **Frontend moderno** | Entrego interfaces reativas com Vue.js, TypeScript e boas práticas de componentização. |

</div>
